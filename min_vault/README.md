# MinVault
![Logo](/min_vault/assets/icon/min_vault.png)
A secure Vault for your Files

## Installationsanweisung 
### Voraussetzungen 
Flutter SDK: >= 3.44.0 (stable) 
Dart SDK: >= 3.12.0 (in der Flutter-SDK-Version enthalten) 
Git 

### Repository klonen 
`git clone https://github.com/xAfterLife/502-EmA` 
### In Projektverzeichnis wechseln 
`cd min_vault` 
### Abhängigkeiten installieren 
`flutter pub get` 
### App ausführen 
`flutter run`
